Projects

My projects are mostly on design, but I’ve also done a few projects involving coding, including this website!

IT Projects

  • Life Expectancy Model

    Me and my team performed an Exploratory Data Analysis with Python libraries, on a Countries Life Expectancy dataset available on Kaggle. From this dataset, we gathered a few insights & recommendations of factors that need to be paid attention to in order to enhance life expectancy.

  • UpTeen

    For the final project of Kode With Klossy’s Web Dev summer program, me and my team created a website to educate teens on financial literacy. I was in charge of the cash flow calculator & financial goals tracker page. I chose this page to work on in order to practice using JavaScript. I currently only have the code to the page that I created!

  • Double Stay Wear Landing Page

    I made this website to submit for Kode With Klossy x Estee Lauder’s in creating a landing page to promote their Double Wear Stay In Place product line. It’s a really simple website, I made it a few days after learning about web development for the first time! I also learned about responsive design while coding this website!

  • FoodRescue

    FoodRescue is a website created for a hackathon submission. Me and my teammates wanted to design a website that could reduce food waste through donations.

  • Learnivity

    Learnivity is a website created for a hackathon submission. Me and my teammates created this website with the goal of how we can make learning more enjoyable & accessible to people from various backgrounds!
